The values above are the observed counts. To find the expected counts, we multiply the total number of trials—in this case, the total number of nuts—by the probabilities of each outcome according to the null hypothesis. Recall that the null hypothesis was:

\begin{equation*} H_0:\ p_1 = 0.40,\ \ p_2 = 0.25,\ \ p_3 = 0.15,\ \ p_4 = 0.10,\ \ p_5 = 0.10 \end{equation*}
