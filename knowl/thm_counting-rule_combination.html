Theorem 2.2.62.

A combination of \(n\) items taken \(r\) at a time is used to count the number of ways to choose \(r\) items from \(n\) possible items if the order in which we select them does not matter. The formula for this combination is:

\begin{equation*} C(n,r) = \frac{P(n,r)}{r!} = \frac{n!}{r!(n-r)!}\text{.} \end{equation*}
