Recall that in SectionÂ 3.2 we introduced the concept of a binomial process. The characteristics of a binomial process were defined to be:

  1. The process consists of a fixed number of trials, each of which had only two possible outcomes: success or failure.

  2. Each trial has the same probability of a success, \(p\text{.}\)

  3. The trials are all independent of each other.

  4. The result variable is the number of trials which result in successes.
