When using large samples to estimate the difference between two population means, we found that we could use the following for the standard deviation of the sampling distribution for the difference between the sample means.

\begin{equation*} \sigma_{\overline{x}_1 - \overline{x}_2} = \sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}\text{.} \end{equation*}
