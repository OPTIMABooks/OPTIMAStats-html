Where then can we get a reliable source of random information? This can actually be a philosophical question. Is anything in the universe truly random, or is everything deterministicâ€”meaning if we know the initial conditions, we can predict exactly what will happen. In this class, we assume that physical phenomena that we observe are, if not random, so complex that they might as well be random. We can gather random numbers then from sources such as:

  • the time between the decay of radioactive material, or

  • time between the observation of cosmic rays, or

  • wind gust speeds and direction.

None of these are terribly practical for us, so instead we use either a pseudo-random number generator on a computer, or a random number table which records digits based on processes similar to those mentioned above.
