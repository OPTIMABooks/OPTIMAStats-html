Definition 6.2.2.

A multinomial process is a random process in which:

  1. there is a fixed number of identical trials, called \(n\)

  2. the outcomes of each trial fall into one of \(k\) possible categories

  3. the probability of the \(i^\text{th}\) category is a fixed probability \(p_i\) with \(p_1 + p_2 + \cdots p_k = 1\text{.}\)

  4. each of the trials is independent

The result of the process is the frequency with which each of the possible categories appear.
